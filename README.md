# Tarek Mahmoud Abdelrady Portfolio

Single-page portfolio website built with React, Vite, and styled-components.

LINK: https://nursery-web-project.vercel.app

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Stack

- React 19
- Vite
- styled-components
- Lucide React

## Notes

- Social links, contact details, experience entries, and certification cards are structured in `src/data/portfolioData.js` for quick editing.
- Project visuals and the Open Graph preview live in `public/images`.
